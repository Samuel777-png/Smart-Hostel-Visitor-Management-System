# Smart Hostel Visitor Management System

This project is a console-based C++ application developed as part of a Software Engineering (SEN) assignment.

## Description
The system records hostel visitor details and determines access approval based on visit purpose.

## Features
- Records visitor details
- Determines access approval
- Displays visit summary
